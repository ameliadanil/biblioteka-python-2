# System biblioteki — programowanie obiektowe

Autor: Amelia Danił  
Numer albumu: 159803

## Opis projektu

Projekt przedstawia system obsługi biblioteki napisany w języku Python z wykorzystaniem programowania obiektowego (OOP).

## Funkcjonalności

- logowanie użytkowników
- przeglądanie katalogu książek
- wypożyczanie książek
- podgląd wypożyczeń
- role użytkowników:
  - czytelnik
  - bibliotekarz
- obsługa próśb o przedłużenie książek
- wyświetlanie wszystkich wypożyczeń przez bibliotekarza

## Zastosowane elementy OOP

- klasy i obiekty
- dziedziczenie
- hermetyzacja
- metoda `__str__`
- klasy pochodne `Reader` i `Librarian`

## Uruchomienie programu

Uruchom plik:

```bash
python biblioteka_oop.py
